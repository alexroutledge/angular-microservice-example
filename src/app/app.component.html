Angular Elements

Web Components: How to create!

Info:

This application creates web components using Angular Elements, which is not released yet, but it works pretty well, there is a branch in angular repository: labs/elements

We have three different web components here:

  • app-like-button - just one simple component
  • app-module-shell - shell for (minimal version, to be integrated in another systems)
  • app-module-root - this is current page that you are viewing (a.k.a. bootstrap page)

We are using routing also,with MockLocationStrategy to make it isolated from main router, when app-module-shell will be used in another application.

Also you can see how those web components can be used: https://angular-webcomponents-usage-sample.now.sh/
